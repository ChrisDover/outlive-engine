# Outlive Engine — Web Frontend

Next.js 16 web frontend for Outlive Engine. See the [root README](../README.md) for full setup instructions.

## Development

```bash
npm run dev       # Start dev server on port 3000
npm run build     # Production build
npm run lint      # Run ESLint
```

## Stack

- Next.js 16, React 19, TypeScript
- Tailwind CSS v4
- NextAuth v4 (JWT strategy)
- Prisma (auth tables only — health data lives in the backend)
